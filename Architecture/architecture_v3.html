Key Research Observations — All 4 Strategies
Strategy A — Color only · 89.06%
  • Highest overall accuracy — color context is the strongest global signal for MobileNetV2
  • Bacterial spot near-perfect at 99.3% — water-soaked lesion color contrast against green leaf is the primary discriminator
  • Healthy leaf 94% — vibrant green color is a strong positive class indicator
  • Spider mites weakest at 77.3% — tiny mite damage patterns masked by background color noise
  • Color-dominant diseases consistently prefer this strategy across all 4 experiments
MobileNetV2 pretrained on color ImageNet naturally leverages color-contrast signals — background carries disease information, not just noise
Strategy B — Segmented only · 86.13%
  • Spider mites jumped 77.3% → 89.3% (+12%) — texture patterns clearer without background
  • Septoria improved 85.3% → 90.7% (+5.4%) — spot shapes more visible on clean black background
  • Yellow Curl highest at 96.7% — curl shape is a pure structural feature independent of color
  • Bacterial spot collapsed 99.3% → 78.7% (−20.6%) — color-contrast signal destroyed by segmentation
  • Leaf Mold dropped 92.3% → 79.0% (−13.3%) — yellow-green color difference was the key discriminator
Background removal is double-edged — helps texture/shape diseases, destroys color-contrast disease signals
Strategy C — Mixed 50/50 · 87.48%
  • Late blight best at 94.7% — consistently improves A→B→C, responds to mixed visual variety
  • Mosaic virus improved to 93.8% — combining both modalities recovers the segmented drop
  • Early blight collapsed 82.0% → 62.0% (−20%) — most critical failure; conflicting representations confused the model
  • Overall 87.48% sits between A and B — random mixing does NOT achieve best of both worlds
  • Early blight's 20% collapse is the strongest motivation for Strategy E class-aware assignment
Random 50/50 mixing fails for color-dominant classes — directly motivates per-class selective assignment (Strategy E)
Strategy D — Fine-tuned · 88.42%
  • Healthy leaf reached 98.7% — highest single-class result across all 4 strategies
  • Mosaic virus reached 96.4% — D uniquely excels here, best result for this class
  • Epoch 1 started at 82.2% val accuracy — proves color knowledge from A successfully retained
  • Catastrophic forgetting minimal — only −0.64% vs Strategy A (99.3% of knowledge retained)
  • Target Spot dropped to 76.7% — worst result for this class across all strategies
  • Bacterial spot dropped to 88.7% (−10.6%) — partial color-contrast forgetting under segmented fine-tuning
Fine-tuning at lr=1e-5 performs sequential domain adaptation with minimal forgetting — per-class trade-offs confirm Strategy E is still needed

🔥 Grad-CAM · Gradient-weighted Class Activation Mapping
Visual Explainability — Why did the model predict this?
Opens the black box by computing gradients of the predicted class score w.r.t. the final convolutional layer's feature maps. Higher gradient = higher pixel importance. Run on all 4 models with the same leaf to show how training data composition changes learned features.
Bacterial spot — % of model attention focused on actual lesion area (higher = better)

  • Mechanism: Backpropagate the class score through the network — compute gradient of class output w.r.t. last conv layer — global-average-pool the gradients — weight feature maps — ReLU the result → heatmap
  • Output: Red zones = model focused here strongly. Blue zones = ignored areas. Overlaid on original leaf image.
  • Research use: Same disease image → 4 heatmaps → visual proof that training strategy changes feature learning. Quantify as attention overlap ratio = (high-attention pixels ∩ disease region) / total high-attention pixels
  • Key classes: Bacterial spot (A should show tight lesion focus), Spider mites (B should show clean texture patterns), Early blight (C should show scattered attention — explaining the 62% collapse)
🌿 Severity Estimator · Disease Regression Module
Severity Estimation — How bad is the disease?
PlantVillage has no severity labels — this module generates new information from existing segmented images. Uses pixel-area analysis to estimate what percentage of the leaf is infected, adding a practical dimension beyond pure classification.
Example output — 3 leaves of the same disease class
Leaf sample 1
Mild
Leaf sample 2
Moderate
Leaf sample 3
Severe
severity_ratio = diseased_pixels / total_leaf_pixels
< 0.15 → "Mild"  ·  0.15–0.40 → "Moderate"  ·  > 0.40 → "Severe"

output: "Early Blight — Moderate (34% leaf affected)"
  • Mechanism: On segmented image, count non-black pixels as total leaf area. Apply HSV/LAB color thresholding to isolate diseased regions (yellowed, browned, spotted). Compute ratio.
  • Type: Regression — predicts a continuous value 0.0–1.0 alongside the classifier's categorical output. The only module in this project that uses regression.
  • Why unique: PlantVillage has no severity annotations — this module creates labeled severity data from unlabeled images. An original research contribution that no existing paper provides for this dataset.
  • Practical value: Farmers need to know HOW BAD the infection is, not just which disease. Severity determines urgency of treatment — pure classification cannot answer this question.
